CSS `margin:auto`的妙用

靠左

.demo-1 {
    width: 100px;
    height: 100px;
}

靠右 (overconstrained)

.demo-2 {
    margin: 0 0 0 auto;
    width: 100px;
    height: 100px;
}

横向居中

.demo-3 {
    width: 100px;
    height: 100px;
    margin: auto;
}

平铺（IE8+）

.demo-4 {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    margin: auto;
}

靠左下角

.demo-5 {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    width: 100px;
    height: 100px;
    margin: auto 0 0 0;
}

    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
